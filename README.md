# 「動手做系列」AWS 上架設 Facebook Messenger Bot

### 活動簡介：
FB message bot 是最近很火熱的話題
如果要做出一個 fb bot
除了要準備程式端的邏輯外
還需要自己架設伺服器
除了需要 handle server 外
還需要處理 api 和 ssl 的部分

這次的動手做
會透過 aws 的 api gateway 和 lambda
用 severless 的方式帶大家部屬自己的 fb bot
有興趣的朋友千萬不要錯過
